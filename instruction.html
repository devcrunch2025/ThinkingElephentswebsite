Package folder "Thinking Elephents Ad Fims" displays files

all .HTML Files - each .html file represents a single template page.

  • index.html - It represent Front page or home page.
  • about.html - It represent about us page.
  • team.html - It represent our team or staff page.
  • faq.html - It represent Frequently ask question page.
  • service.html - It represent service page.
  • portfolio.html - It represent Portfolio gallery page.
  • blog.html - It represent blog page.
  • contact.html - It represent contact us page.

'asset' folder Files-

  • css - style.css, animate.css, aos.css
  • img - all images
  • js - aos.js, count-timer.js, gallery.js, MobileViewBtn.js, scroll.js, testimonial-active-item.js
  • video - presentation.mp4 (presentation clip)

How to edit text

Templates are HTML based templates so all content is stored and could be accessed though the .html files.

  • Open your template package and go to the folder "Thinking Elephents".
  • Then open index.html file with your HTML editor.
  • You can use the search tool CTRL+F to find any text you need.
  • Edit text in HTML editor. Save the file and open it with your Browser to see the changes.

How to edit images

Open the template folder, go to 'asset/img' directory. You can upload your images with the same name and extension to replace the default ones.

Details about CSS files

  • style.css - main CSS file. All Style Sheets is changed by this file
  • animate.css - all animation is created by this file
  • aos.css - on scroll animation is created by this file

Details about JS files

  • aos.js - used to create on scroll animation
  • count-timer.js - used for number count animation
  • gallery.js - used to create portfolio gallery with filter and animation
  • MobileViewBtn.js - used to change menu button in small screen
  • scroll.js - Change background color of Fixed Top menu on scroll
  • testimonial-active-item.js - In customer review section of front page cutomer image indicate the active comment.
  • nav-video-transition.js - optional navigation transition overlay; plays `asset/video/navigation-video.mp4` (fallback: `home.mp4`). Configure duration in `asset/js/nav-video-transition.js` (TIMEOUT ms).
